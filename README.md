# Complexe Scolaire Privé Les Dauphins Bleus

Site web officiel du Complexe Scolaire Privé Les Dauphins Bleus à Niamey, Niger.

## Structure du projet

```
franck/
├── css/
│   └── style.css          # Feuille de style principale
├── images/                # Toutes les images du site
├── index.html             # Page d'accueil
├── apropos.html          # Page À propos
├── contact.html          # Page Contact
├── galerie.html          # Page Galerie
├── politique.html        # Politique de confidentialité
├── cgu.html              # Conditions Générales d'Utilisation
└── README.md             # Ce fichier
```

## Technologies utilisées

- **HTML5** : Structure sémantique
- **CSS3** : Styles et mise en page responsive
- **JavaScript** : Fonctionnalités interactives (minimal)
- **Font Awesome** : Icônes

## Fonctionnalités

- ✅ Design responsive (mobile, tablette, desktop)
- ✅ Navigation fluide entre les pages
- ✅ Galerie photo avec lightbox
- ✅ Formulaire de contact fonctionnel (Formspree)
- ✅ Bouton retour en haut
- ✅ Animations au survol
- ✅ Carte Google Maps intégrée
